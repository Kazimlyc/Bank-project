# Bank Project
Java-based desktop banking application implementing core account operations with a GUI.

## Overview
A Java desktop application built in NetBeans to practice OOP by simulating common banking workflows such as authentication, transfers, deposits/withdrawals, and bill payments.

## Features
- User registration & login
- Deposit / withdrawal
- Account-to-account transfer
- Bill payments
- GUI screens for user flow

## Tech Stack
- Java
- Swing (GUI)
- MySQL
